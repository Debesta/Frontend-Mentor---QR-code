### What I learned

First time I really see how paddings nad margins works together.

```scss

	width: 270px;
	margin: 15px 0px 20px;
	border-radius: 10px;
}
h1 {
	padding: 0 20px;
	margin-bottom: 10px;
	font-size: 20px;
	color: $dark-blue;
}
p {
	padding: 0 20px;
	margin-bottom: 30px;
	font-size: 15px;
	color: $grayish-blue;
}
```
